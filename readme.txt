== Description ==
Floating Survey is a powerful WordPress plugin that allows you to easily add surveys to your website. With this plugin, you can create multistep surveys, collect user feedback, and gain valuable insights for your business or personal use.

The plugin provides a user-friendly interface to create and manage surveys directly from the WordPress admin panel. You can configure the survey settings, add questions, define multiple steps, and customize the appearance of the floating button.

Key features include:
- Create and manage surveys from the plugin settings panel.
- Display a floating button on the bottom left or right of all pages on your website.
- Users can click on the floating button to view a list of available surveys.
- Surveys open in a popup modal for a seamless user experience.
- Multistep surveys allow users to provide contact information and answer questions.
- Submissions are stored in the database for easy access and analysis.
- Admins can view survey submissions and analyze survey data from the plugin panel.
- Free and paid plans available. The free plan allows users to add one survey.

== Installation ==
1. Upload the  `floating-survey`  folder to the  `/wp-content/plugins/`  directory or install the plugin directly from the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Floating Survey' settings page in the WordPress admin panel to configure your surveys and customize the floating button appearance.
4. Create your surveys by adding questions and defining the survey steps.
5. Customize the thank you email template that users will receive upon submitting the survey.
6. Start collecting valuable feedback and analyzing survey results!

== Frequently Asked Questions ==
= Can I add multiple surveys to my website? =
Yes, you can create multiple surveys and configure their settings individually. However, please note that the free plan allows adding only one survey.

= Can I customize the appearance of the floating button? =
Absolutely! The plugin provides options to customize the floating button's position (bottom left or right) and style to match your website's design.

= Can users see the survey on all pages of my website? =
Yes, the floating button appears on all pages of your website, allowing users to access the surveys at any time.

= How can I view and analyze survey submissions? =
All survey submissions are stored in the plugin's database. You can easily access and analyze them from the plugin panel in the WordPress admin area.
